 Say cheese!

Accesses the webcam with getUserMedia(). 
You can apply different CSS filters to the webcam stream in real time.
Take still pictures.
Re-arange them with drag and drop.

This is a simplified version of the application that is on GitHub. That version can post the still image collage to Facebook.

Works On My Machine (TM) in Chrome, Opera and Firefox.

Ost means cheese in Swedish.
